#  Blog Total

O **Blog Total** é um projeto pessoal onde compartilho ideias, estudos e projetos relacionados à tecnologia, programação e curiosidades do mundo dev.

A proposta é manter um espaço simples, direto e organizado, com conteúdos no estilo clássico de leitura.

---




## 🛠 Tecnologias utilizadas

* HTML5 (semântico)
* CSS3 (responsivo)
* JavaScript (ES6+)
* Bootstrap 5

---

## 📂 Estrutura do projeto

```bash
/
├── index.html
├── README.md
└── assets/
    ├── css/
    ├── js/
    ├── img/
    └── pages/
```

---

## ⚙️ Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/seuusuario/blogtotal.git

# Abra o arquivo
index.html
```

---

## 📌 Funcionalidades

* Navegação entre páginas
* Estrutura de artigos organizada
* Layout responsivo
* Botão de voltar ao topo

---

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de:

* Praticar desenvolvimento front-end

* Evoluir a estrutura de projetos reais
