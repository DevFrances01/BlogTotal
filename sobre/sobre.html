Quem Sou Eu ?

Este projeto começou como uma ideia simples de bloco de notas, mas foi crescendo até se transformar em um blog. Ainda está em construção, mas já mostra como vai ficar no futuro.

O Objetivo

Compartilhar experiências, ideias e aprendizados, além de servir como um espaço para praticar desenvolvimento web. Aos poucos, o sistema vai ganhar novos recursos, como área de comentários, login com código e perfis de usuários.

Próximos Passos

  • Área de comentários funcional
  • Sistema de login simplificado
  • Perfis com foto personalizada
  • Banco de dados para salvar tudo
